# Библиотеки

Каталог предназначен для общих библиотек монорепозитория. На этапе 0
зафиксирована только `shared`, потому что именно она упоминается в архитектуре
как общий слой сервисов.

| Библиотека | Назначение |
|------------|------------|
| [shared](shared/) | Tenant context, error envelope, audit utilities, общие модели и helpers. |
