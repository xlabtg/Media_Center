# Content Generator & Link Router

**Статус:** каркас сервиса, реализация запланирована в этапе 2.

## Назначение

Content Generator & Link Router (CGLR) генерирует контент по шаблонам,
маршрутизирует ссылки L1/L2/L3 и передаёт готовые материалы в Unified
Messenger Adapter для публикации.

## Границы ответственности

- хранит и применяет шаблоны генерации контента;
- рендерит шаблоны через sandboxed Jinja2 и валидирует результат по длине и
  обязательным блокам;
- выбирает tenant-aware маршруты ссылок;
- валидирует ограничения площадок перед отправкой в адаптер публикации;
- регистрирует вклад через Contribution Ledger, не владея расчётом баллов.
